## 📌 Overview

This repository documents my learning journey through the TechCrush AI/ML Engineering Bootcamp. The program covers foundational to advanced concepts in Artificial Intelligence and Machine Learning, with a strong emphasis on practical, hands-on projects.

## 📖 Key Concepts Covered

### Machine Learning
- Supervised vs Unsupervised Learning
- Classification (KNN, Logistic Regression, SVM)
- Regression (Linear, Polynomial, Lasso, Ridge)
- Clustering (K-Means, Hierarchical, GMM)
- Ensemble Methods (Random Forest, Gradient Boosting)
- Train/Test Split, Data Leakage, Cross-Validation

### Deep Learning
- Neural Networks (Perceptrons, Activation Functions, Layers)
- Overfitting & Regularization (Dropout)
- MNIST Handwriting Recognition

### Computer Vision
- Convolutional Neural Networks (CNNs)
- Filters, Kernels, Feature Maps
- Transfer Learning (VGG16, ResNet)
- Object Detection & Image Segmentation

### Data Preprocessing
- EDA (Exploratory Data Analysis)
- Missing Data Imputation
- Outlier Detection (IQR Rule)
- Feature Engineering & Selection
- One-Hot Encoding, Label Encoding
- Standardization & Normalization
